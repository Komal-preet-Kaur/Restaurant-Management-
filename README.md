# 🍽️ DineDelight - Restaurant Management System

A full-stack web application where users can explore restaurants, make reservations, and manage bookings easily. Built using modern web technologies with a clean and responsive UI.

---

## 🚀 Tech Stack

**Frontend:** React (Vite), TypeScript, CSS
**Backend:** Node.js, Express.js
**Database:** MongoDB (Mongoose)
**Authentication:** Sessions + bcrypt

---

## ✨ Features

* 🔐 User Signup & Login
* 🍴 Browse & Explore Restaurants
* 📅 Table Reservation System
* 📋 User Dashboard (view/manage bookings)
* 📩 Contact Form
* 📱 Fully Responsive UI

---

## 📂 Project Structure

```
├── frontend/        # React frontend
├── api/             # API routes
├── models/          # Database schemas
├── middlewares/     # Backend middlewares
├── server.js        # Backend entry point
```

---

## ⚙️ Setup

### 1. Clone the repository

```
git clone https://github.com/Komal-preet-Kaur/Restaurant-Management-
cd Restaurant-Management-
```

### 2. Install dependencies

**Backend**

```
npm install
```

**Frontend**

```
cd frontend
npm install
```

---

### 3. Run the project

**Backend**

```
npm start
```

**Frontend**

```
npm run dev
```

---

## 🔗 API Routes (Basic)

* `/api/auth` → Login / Signup
* `/api/reservations` → Manage bookings
* `/api/contact` → Contact form
